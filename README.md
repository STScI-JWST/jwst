JWST Calibration Pipeline
=========================

Installing
----------

For developers:

    % git clone https://github.com/STScI-JWST/jwst.git

    % cd jwst

    % python setup.py install

    or

    % python setup.py develop

Using
-----

Documentation (built nightly) is available here:

http://ssb.stsci.edu/doc/jwst_dev/
