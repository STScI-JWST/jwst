===========
AMI Analyse
===========

This step applies the Lacour-Greenbaum (LG) image plane modeling 
algorithm to a NIRISS Aperture Masking Interferometry (AMI) image. The 
routine computes a number of parameters, including a model fit (and residuals) 
to the image, fringe amplitudes and phases, and closure phases and amplitudes.
